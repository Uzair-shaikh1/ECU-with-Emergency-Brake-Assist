# Raspberry Pi Dashboard

This folder contains the Python dashboard code used for displaying and logging the Safety Supervisor ECU states.

## Purpose

The dashboard shows the current safety condition of the system such as:

- Normal
- Warning
- Emergency

It is used on Raspberry Pi 4B with display support for real-time monitoring of the project.
